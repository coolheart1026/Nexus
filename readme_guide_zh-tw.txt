手動輸入模式（Manual）：
1. 把號碼粘貼在名爲“ID”的輸入欄裏
2. 選擇打包成ZIP格式或是文件夾（FOLDER）格式
3. 選擇保存位置
4. 點擊開始下載後靜候下載完畢，有實時讀取條

剪貼板偵測模式 （Clipboard）：
1. 選擇打包成ZIP格式或是文件夾（FOLDER）格式
2. 選擇保存位置
3. 複製格式為“#123456”的號碼自動下載， 有實時讀取條

To_PDF_Converter是一個圖片轉換程序(可以複製粘貼到需要的位置)。它會讀取同層所有文件夾裏的圖片(只讀取下一層)然後根據圖片名稱順序合并在一起轉換成PDF格式。只支援"jpg", ".jpeg", ".png", ".webp", ".bmp", ".tiff", ".tif", ".gif"的格式轉換。
